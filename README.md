# Student Learning Behaviour Analysis using Machine Learning

## 📌 Project Overview
This project predicts student learning behaviour using Machine Learning techniques. 
It analyzes student data and provides predictions based on behavioral characteristics.

## 🚀 Technologies Used
- Python
- Flask
- HTML, CSS
- Machine Learning
- Pandas
- NumPy
- Scikit-learn
- SQLite Database

## 📊 Features
- Student login & signup system
- Data preprocessing
- ML model prediction
- Web-based interface using Flask
- Trained model saved as model1.sav

## 🧠 Machine Learning
The model is trained using student dataset and saved using pickle (.sav file).
The application loads the trained model and predicts student learning behaviour.

## 📂 Project Structure
- app.py → Main Flask Application
- templates/ → HTML files
- static/ → CSS, JS files
- model1.sav → Trained ML model
- Dataset/ → Dataset used for training
- signup.db → Database file

## ▶️ How to Run the Project

1. Install requirements:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/
